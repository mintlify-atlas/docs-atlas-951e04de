---
title: "Monitorian: Multi-Monitor Brightness Control for Windows"
description: "Monitorian adjusts brightness and contrast on multiple monitors — including external DDC/CI displays — from the Windows system tray. No OSD menus needed."
---

Monitorian lets you take control of every display connected to your Windows PC. Whether you have one monitor or several, you can fine-tune brightness and contrast from the system tray — no OSD menus required. External monitors are supported via DDC/CI, and all adjustments can be made individually or simultaneously.

## Get up and running

<Steps>
  <Step title="Check requirements">
    Ensure you're running Windows 7 or newer with .NET Framework 4.8. For external monitors, DDC/CI must be enabled on the display. See [Requirements](/requirements).
  </Step>
  <Step title="Install Monitorian">
    Download from the [Microsoft Store](https://www.microsoft.com/store/apps/9nw33j738bl0), use `winget install Monitorian -s msstore`, or grab the [direct installer](https://github.com/emoacht/Monitorian/releases). See [Installation](/installation).
  </Step>
  <Step title="Adjust brightness">
    Click the Monitorian icon in your system tray. Drag the slider for any detected monitor to change brightness. Use the unison toggle to move all monitors together.
  </Step>
  <Step title="Explore add-on features">
    Unlock hot keys, command-line control, and automation commands with the Microsoft Store subscription add-on. See [Add-on Features](/addons/hotkeys).
  </Step>
</Steps>
